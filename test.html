CleanURLs Extension Test Page

This page contains links with tracking parameters to test the CleanURLs extension.

Test Links:

Link with UTM parameters Link with Facebook tracking Link with Google tracking Link with Microsoft tracking

Instructions:

  1. Install the CleanURLs extension
  2. Open this page
  3. Open DevTools (F12) → Console tab
  4. Reload this page to see CleanURLs debug messages
  5. Right-click on any link and "Copy link address"
  6. Check if tracking parameters are removed
  7. Open the extension popup to see cleaned links counter

Debug Info:

Check the browser console for CleanURLs debug messages that will show:

  • ✅ Rules loaded from storage
  • ✅ Links found and processed
  • ✅ Parameters removed
  • ❌ Any errors encountered
